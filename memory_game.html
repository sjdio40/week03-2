🚀 프로젝트 개요

"아두이노가 내는 순서를 기억하고 따라 해보세요!"

목표

아두이노의 기본 디지털 입출력(I/O) 제어를 이해하고, 이를 활용하여 누구나 즐길 수 있는 인터랙티브 게임을 제작하는 것을 목표로 합니다.

게임 규칙

  1. 시작: 아무 버튼이나 눌러 게임을 시작합니다.
  2. 기억: 5개의 LED가 무작위 순서로 깜빡입니다. 이 순서를 잘 기억합니다.
  3. 입력: 기억한 순서대로 LED에 해당하는 버튼을 누릅니다.
  4. 성공/실패: 순서를 모두 맞히면 성공 멜로디가, 틀리면 실패 효과음이 나옵니다.

🛠️ 사용된 부품 및 회로도

핵심 부품 리스트

  • 제어 장치: 아두이노 Uno R3
  • 출력 장치: LED 5개, 수동 부저 1개
  • 입력 장치: 푸시 버튼 5개
  • 기타: 브레드보드, 점퍼 와이어, 220Ω 저항 5개

회로 구성

버튼 연결 시, 아두이노의 내부 풀업(Pull-up) 저항을 사용하여 별도의 저항 없이 회로를 간소화했습니다. LED는 2~6번, 버튼은 8~12번, 부저는 7번 핀에 연결합니다.

💻 코드 핵심 로직 (Flowchart)

프로그램은 `생성` → `출력` → `입력` → `판단` 의 명확한 흐름을 가집니다.

[게임 시작]
      ↓
[무작위 순서 생성]
      ↓
[LED로 순서 출력]
      ↓
[사용자 버튼 입력 대기]
      ↓
[입력 순서 확인] ─── (틀림) ───▶ [실패 효과음]
      │
 (맞음, 진행중)
      ↓
[5개 모두 성공] ───▶ [성공 멜로디]

주요 함수

  • generateSequence(): 게임의 정답 순서를 만듭니다.
  • displaySequence(): 생성된 순서를 LED 불빛으로 보여줍니다.
  • readButtons(): 버튼 중복 입력을 방지(디바운싱)하며 입력을 감지합니다.
  • playSuccessMelody(): 성공 시 축하 노래를 재생합니다.

🎉 완성 및 발전 방향

프로젝트 결과

LED와 버튼을 이용한 기억력 게임이 정상적으로 작동하며, 성공 및 실패에 대한 명확한 시청각적 피드백(LED, 부저)을 구현했습니다.

어려웠던 점 및 해결

문제: 버튼을 한 번 눌러도 여러 번 입력되는 '채터링' 현상 발생
해결: 간단한 디바운싱(Debouncing) 로직을 추가하여 안정적인 입력을 구현했습니다.

향후 발전 방향

  • 난이도 조절: 성공 시 순서가 1개씩 늘어나는 '단계별 모드' 추가
  • 점수 시스템: 7세그먼트나 LCD를 추가하여 현재 점수 표시
  • 타임 어택 모드: 입력 시간제한을 두어 긴장감 부여
